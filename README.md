# Youtube Workflow

## Installation

1. Download the latest `alfred-youtube.alfredworkflow`
2. Open from Finder to add to Alfred
3. Enjoy!

## Usage
`yt`: Play or pause the current YouTube video in Google Chrome.

Suggestions welcome!
